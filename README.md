# RunPod ComfyUI Ephemeral Template

Template per avviare ComfyUI su RunPod in modalit√† ephemeral (senza persistenza).

## Caratteristiche
- Setup automatico di ComfyUI
- Download automatico dei modelli all'avvio
- Nessun volume persistente richiesto
- Pronto in 2-3 minuti

## Come usare
1. Vai su RunPod.io
2. Crea un nuovo template usando questo repository
3. Avvia un pod e attendi il caricamento
4. Accedi a ComfyUI sulla porta 8188

## Modelli inclusi
- Stable Diffusion 1.5
- VAE standard

## Personalizzazione
Modifica il file `startup.sh` o `modelli.txt` per aggiungere i tuoi modelli preferiti.
